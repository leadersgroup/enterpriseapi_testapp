📋 API Endpoints

GET Get Pricing Information

Retrieve pricing for a specific state and county. If county pricing is not found, returns default state pricing.

GET List All Orders

Retrieve a list of all orders in the system.

GET Get Specific Order

Retrieve details for a specific order by ID.

POST Create New Order

Create a new deed recording order. Server automatically determines pricing based on user plan.

POST Register Webhook

Register a webhook URL to receive notifications for order events.
